# Unnamed Messenger App

A base web messenger starter built with React, Vite, Express, and Socket.IO.

## Structure

- `client/` — React frontend powered by Vite
- `server/` — Express backend with Socket.IO real-time messaging

## Run locally

1. Install dependencies:
   ```bash
   npm install
   npm run install:all
   ```
2. Start development servers:
   ```bash
   npm run dev
   ```

The frontend runs on `http://localhost:5173` and connects to the backend socket server on `http://localhost:4000`.
